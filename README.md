### Dreamstreamv2
Upgrade tx onchain 
## DreamStream PRO+ 

## How to deploy
1. Copy all files into repo root.
2. Push to GitHub and connect to Vercel, or deploy to static host.
3. Open site (https) and click **Connect MetaMask**.
4. Allow network switch to Somnia Testnet (chainId `0xC487`).
5. Play Pacman (hybrid), submit score, claim reward.

## Files
- index.html — Dashboard parent
- pacman_onchain.html — Onchain only
- pacman_iframe.html — iframe legacy (postMessage)
- pacman_hybrid.html — Hybrid (on-chain + postMessage)
- config.js — Somnia + contract config
- web3.js, transaction.js — Ethers helpers
- stream_adapter.js — Somnia SDS adapter
- somnia_chain.js — add/switch chain helper
